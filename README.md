## 🎯 Problem Statement
The objective of this project is to predict medical insurance charges based on various features such as age, BMI, smoking habits, and region using Linear Regression.

---

## 📊 Dataset Description
We used the Medical Cost Personal Dataset.

Features:
- age: Age of the person
- sex: Gender (male/female)
- bmi: Body Mass Index
- children: Number of children
- smoker: Whether the person smokes or not
- region: Residential area

Target Variable:
- charges: Medical insurance cost

---

## 🧹 Data Preprocessing Steps
- Checked for missing values (no missing values found)
- Converted categorical variables into numerical using encoding
- Scaled numerical features using StandardScaler
- Prepared dataset for machine learning model

---

## 🤖 Model Used
We used Linear Regression model from sklearn.

Steps:
- Split data into training (80%) and testing (20%)
- Trained the model using training data
- Predicted values using test data

---

## 📈 Model Evaluation
We evaluated the model using the following metrics:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

These metrics help us understand the accuracy and performance of the model.
